Fonts

Get a preferred font. I personally go for Oxygen Mono

Shortcuts

Remap Caps Lock to F13. Restart is required to take effect

Get Keypirinha

Open settings file by right clicking on Keypirinha icon in the notification area, and choose "Configure Application"

Change default settings to:

[app]
hotkey_run = F13
launch_at_startup = yes

Works greati with Search Everything with the built-in Keypirinha plugin
