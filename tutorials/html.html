HTML cheat sheet

A basic skeleton
<!DOCTYPE html>
<head>
 <title>Page title</title>
<html>
<body>
</body>
</html>

opening bracket - <html>
closing bracket - </html>

<br> or <br /> is a self-closing tag. Both formats are okay
<tagName attributeName='atributeValue'>Content>/tagName>

Escape sequences for HTML symbol
< - &lt;
> - &gt;
& - &amp;
' - &apos;
" - &quot;
TAB - &emsp;

<em>TEXT</em>
<strong>TEXT</strong>
Linebreak - <br>

<h1>HEADING<h1>

<h2>HEADING<h2>

<h3>HEADING<h3>

<h4>HEADING<h4>

<h5>HEADING<h5>
<h6>HEADING<h6>

Lists
Ordered list
<ol>

  1. <li>Item A</li>
  2. <li>Item B</li>
</ol>

Unordered list
<ul>
  • <li>Item A</li>
  • <li>Item B</li>
</ul>

Links
<a href="DESTINATION">TEXT</a>

Image
<img src="DESTINATION" width="INT" height="INT" alt="Text alternative">

<table>
 <tr>
  <td>Row 1, cell 1</td>
  <td>Row 1, cell 2</td>
 </tr>
 <tr>
  <td>Row 2, cell 1</td>
  <td>Row 2, cell 2</td>
 </tr>
</table>

|Row 1, cell 1| |Row 1, cell 2|
|Row 2, cell 2| |Row 2, cell 1|

Forms
get is used for shorter, non-sensitive information. Uses web adress
post is used for longer, more secure submission. More or less invisible

<form action="SCRIPT_NAME.php" method="post OR get">
 <input value="INITIAL TEXT" name=fieldName>
 

 <input type="password">
 

 <input type="checkbox" checked>
 

 <input type="radio" name="GROUP_NAME" checked>
 <input type="radio" name="GROUP_NAME">;
 

 <input type="submit" value="BUTTON_TEXT">
 

 <textarea rows="NUMBER_OF_ROWS" cols="NUMBER_OF_COLUMNS">INITIAL TEXT</textarea<
 
 <select>
  <option>Option 1</option>
  <option selected>Option 2</option>
  <option value="Option 3">Option 3</option>
 </select>

</form>
