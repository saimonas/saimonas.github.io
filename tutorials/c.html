Chapter 11: Turn your functions up to 11

Function name is a pointer. Function names are pointer variables. There are is no special data type for function pointers. To create function pointer:

returnType (*functionAlias) (paremeterType1, parametertype2);

functionAlias = functionName

To call the function, just the function with the alias

Example:

int (*warp_fn)(int);

warp_fn = go_to_warp_speed;

warp_fn(4);
